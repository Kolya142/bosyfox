Search query

!! - raw google search
!u - raw URL
!p - load Plugin
!g - Google search
!d - Duckduckgo search
!y - Yandex search
!f - builtin Function: !fbosycad - open BOSYCAD; !freload - reload browser; !fhelp - open this page
!t - google Translate !tFROM2TO. Example: !tru2en
default case - just search in google

Plugin API

export function load(plug_filename) {return {...};} - Plugin init function
.keyDown(key)
.tick()
.doSearch(query) - Plugin any search
.doFunc(query) - Plugin !f search
.plugid - Plugin ID, format DOMAIN.COMPANY.APPID
